# Public Health Trend Intelligence — COVID-19 ML Forecasting + Cortex AI

> **TAMU CSEGSA x Snowflake Hackathon 2026** | Track C — AI for Social Good  
> Team of 3 | 24-Hour Sprint | April 9–10, 2026

## What It Does

A public health intelligence system that combines **Snowflake ML Forecasting** with **Cortex AI** to help non-technical health officials understand and act on COVID-19 trends across 14 countries.

- **ML Forecasting**: Per-country 30-day case projections with confidence intervals using Snowflake's native ML Forecasting API with exogenous features (vaccination rates, death trends)
- **Cortex AI Summarization**: Plain-English outbreak narratives, policy recommendations, and explainability narratives generated by Snowflake Cortex `COMPLETE()`
- **Interactive Dashboard**: 9-tab Streamlit app with forecast charts, risk tiers, country comparisons, leaderboards, and AI-generated insights

## Setup Instructions

### Prerequisites
- Snowflake trial account with Marketplace access and Cortex credits
- Warehouse: `COMPUTE_WH` (or adjust in scripts)

### Step-by-step

1. **Get Marketplace data**: Log into Snowflake → Data Products → Marketplace → Search "COVID-19 epidemiological" → Click **Get** on the Starschema dataset

2. **Create schemas**: Open a SQL Worksheet → paste and run `scripts/01_setup_database.sql`

3. **Run the pipeline in order**:
   ```
   01_setup_database.sql          → SQL Worksheet
   step1_ingest_and_profile.sql   → SQL Worksheet
   step2_feature_engineering_pyworksheet.py → Python Worksheet (set DB: HACKATHON, Schema: INT)
   step3_ml_forecasting.sql       → SQL Worksheet (run phase by phase)
   04_cortex_summarization.sql    → SQL Worksheet (run section by section)
   ```

4. **Launch dashboard**: Projects → Streamlit → + Streamlit App → DB: HACKATHON, Schema: MARTS → Paste `05_streamlit_dashboard.py` → Run

## Key Features

### Feature Engineering (23 features)
- **Rolling averages**: 7-day and 14-day case/death/vaccination averages
- **Doubling time**: Days to double cumulative cases at current growth rate
- **Vaccination rate lags**: 14/21/28-day lagged vaccination coverage (biological delay modeling)
- **Explainability features**: Case-death ratio, surge flags, vaccination impact proxy

### ML Forecasting
- Snowflake ML Forecasting API with **exogenous features** (vaccination lags, death trends)
- Per-country multi-series model across 14 countries
- 30-day holdout evaluation with **sMAPE, WMAPE, MAE** metrics
- 95% prediction intervals for uncertainty quantification
- Feature importance extraction for explainability

### Cortex AI (3 types of LLM output)
1. **Outbreak narratives**: WHO-style situation briefings per country
2. **Policy recommendations**: 3-point actionable policy advice per country
3. **Explainability narratives**: Plain-English explanations of why the model predicts each trajectory

### Data Quality & Fairness
- Batch-reporting detection (countries reporting weekly, not daily)
- Zero-case day analysis and geographic bias documentation
- End-of-pandemic surveillance decline noted
- Turkey excluded from ML (100% zero cases in training window)
